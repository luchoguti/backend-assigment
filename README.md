## Description

Documentation of knowledge tests for the company DocRed.

## Pre - Installation

The system should have installed:

[NodeJs](https://nodejs.org/es/) -
[NPM](https://www.npmjs.com/) -
[Docker](https://www.docker.com/) -
[NestJs](https://nestjs.com/). 

## Installation

```bash
$ git clone https://github.com/luchoguti/backend-assigment
$ cd backend-assigment
$ npm install
```

## Running the app

```bash
# database
$ cd /database
$ docker-compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
